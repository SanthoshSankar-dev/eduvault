## 🔄 Application Flow

```
1. Visit http://localhost:5000/login
2. Register account or login with credentials
3. Redirected to /dashboard
4. Browse options:
   - Click "Upload Notes" → /upload-notes
   - Click "View Notes" → /view-notes
   - Click subject cards → /view-notes
5. Upload or download notes
6. Return to dashboard
```
